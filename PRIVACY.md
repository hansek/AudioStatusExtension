# Privacy Policy

Audio Status for Command Palette does not collect, store, transmit, sell, or share personal data.

The extension reads local Windows audio endpoint information only to display the current input and output devices and to switch the default audio device when requested by the user. This information remains on the user's device and is not sent to Jan Tezner, Microsoft, or any third party by this extension.

The extension does not use analytics, advertising, telemetry, tracking cookies, accounts, or cloud services.

For support, use the project issue tracker:
https://github.com/hansek/AudioStatusExtension/issues
